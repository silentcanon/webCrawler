This is the demo page for the parallel web crawler written in Java. The source codes could be found here.

It crawlers a certain number of pages from this starting page. The source code of the web crawler could be found here. Click buttons above for the demo.

This web page is build using Bootstrap and D3.js

This demo represents the crawling results.

Each node represents one single resource on the Internet. Each arrow represents the link between online resources. The resource could be html files, css files, pdf files, images and etc.

In this page, nodes in dark blue represent web pages (html files); Orange for image files such as png, gif and jpg. Light blue for others such as pdf.

In the mathematical theory of directed graphs, a graph is said to be strongly connected if every vertex is reachable from every other vertex. Click here for more details.

For the nodes and links represent the web resources, it is interesting to calculate the strongly connected components to illustrate the structure of the web. In the graph on the right, nodes with the same color belong to one connected component.
